# E-SKOLAR

#### DEPENDENCIES
- php
- composer
- local database (mysql || xampp || lamp || postgresql || any)
- nodejs

#### SETUP
- ##### Tailwindcss
  - run ``` npm install ```
- ##### Laravel
  - run ``` composer update ```
  - rename ``` .env.example ``` file to ``` .env ```
  - configure ``` .env ``` file
  - run ``` php artisan migrate ```
  - run ``` php artisan key:generate ```
  - run ``` php artisan serve ```
